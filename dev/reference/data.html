PowerModelsONM.convert_storage! — Method
convert_storage!(nw::Dict{String,Any})

Helper function for PowerModelsProtection fault studies to convert storage to generators in a subnetwork; PowerModelsProtection currently does not support storage object constraints / variables, so this is a workaround until those constraints/variables are added.

This works on ENGINEERING subnetworks (not multinetworks).
